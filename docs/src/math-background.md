# Mathematical Background

This page provides the mathematical foundations for numerical semigroups, numerical sets, and integer partitions.

## Numerical Semigroups

### Definition

A **numerical semigroup** is a subset $S \subseteq \mathbb{N}_0 = \{0, 1, 2, 3, \ldots\}$ satisfying:

1. **Closure under addition**: If $a, b \in S$, then $a + b \in S$
2. **Contains zero**: $0 \in S$
3. **Finite complement**: $\mathbb{N}_0 \setminus S$ is finite

The elements not in $S$ are called **gaps**.

### Example

Consider $S = \{0, 3, 6, 8, 9, 10, 11, 12, \ldots\}$:

- Contains 0
- Closed under addition: $3 + 3 = 6 \in S$, $3 + 6 = 9 \in S$, etc.
- Finite gaps: gaps are $\{1, 2, 4, 5, 7\}$

This is generated by $\{3, 8\}$, written $S = \langle 3, 8 \rangle$.

### Key Invariants

#### Frobenius Number

The **Frobenius number** $g(S)$ is the largest integer not in $S$.

**Example:** For $S = \langle 3, 8 \rangle$, we have $g(S) = 7$ since 7 is the largest gap.

**Properties:**
- Always exists for proper numerical semigroups
- Determines when all integers $\geq g(S) + 1$ are in $S$
- Named after the "coin problem" (Frobenius problem)

#### Genus

The **genus** $g$ is the number of gaps: $g = |\mathbb{N}_0 \setminus S|$.

**Example:** For $S = \langle 3, 8 \rangle$, genus is 5 (gaps: 1, 2, 4, 5, 7).

**Properties:**
- $g = 0$ if and only if $S = \mathbb{N}_0$
- Related to Frobenius number: $g \leq g(S) + 1$
- For 2 generators: $g = \frac{(a-1)(b-1)}{2}$ when $\gcd(a,b) = 1$

#### Multiplicity

The **multiplicity** $m$ is the smallest positive integer in $S$.

**Example:** For $S = \langle 3, 8 \rangle$, multiplicity is 3.

**Properties:**
- Always exists (since $S$ is not just $\{0\}$)
- Divides infinitely many elements of $S$
- $m \leq g(S) + 1$

### Apéry Set

For an element $n \in S$, the **Apéry set** $\text{Ap}(S, n)$ consists of the smallest elements of $S$ in each residue class modulo $n$:

$$\text{Ap}(S, n) = \{s \in S : s - n \notin S\}$$

**Example:** For $S = \langle 3, 5 \rangle$ and $n = 3$:
- $\text{Ap}(S, 3) = \{0, 5, 10\}$ (one element per residue class mod 3)

**Properties:**
- $|\text{Ap}(S, n)| = n$
- Determines the semigroup completely
- Essential for many algorithms

### Minimal Generating Set

A **generating set** $\{a_1, \ldots, a_k\}$ for $S$ is minimal if no element can be removed while still generating $S$.

**Example:** 
- $\langle 3, 5 \rangle = \{0, 3, 5, 6, 8, 9, 10, 11, \ldots\}$
- $\{3, 5\}$ is minimal
- $\{3, 5, 6\}$ is not (since $6 = 3 + 3$)

**Properties:**
- Always exists and is unique
- Elements are called **minimal generators**
- Can be computed from the Apéry set

## Integer Partitions

### Definition

An **integer partition** of $n$ is a way of writing $n$ as a sum of positive integers:

$$n = \lambda_1 + \lambda_2 + \cdots + \lambda_k$$

where $\lambda_1 \geq \lambda_2 \geq \cdots \geq \lambda_k > 0$.

We write $\lambda = (\lambda_1, \lambda_2, \ldots, \lambda_k)$ and say $\lambda \vdash n$.

### Ferrers Diagram

The **Ferrers diagram** of $\lambda = (\lambda_1, \ldots, \lambda_k)$ is a left-justified array of boxes:
- Row $i$ has $\lambda_i$ boxes
- Rows aligned to the left

**Example:** For $\lambda = (5, 4, 3, 1)$:

```
■ ■ ■ ■ ■
■ ■ ■ ■
■ ■ ■
■
```

### Conjugate Partition

The **conjugate** $\lambda'$ is obtained by reflecting the Ferrers diagram across the main diagonal.

**Example:** $(5, 4, 3, 1)' = (4, 3, 3, 2, 1)$:

```
Original:      Conjugate:
■ ■ ■ ■ ■      ■ ■ ■ ■
■ ■ ■ ■        ■ ■ ■
■ ■ ■          ■ ■ ■
■              ■ ■
               ■
```

**Properties:**
- $(\lambda')' = \lambda$ (involution)
- $\lambda'_j$ = number of parts $\geq j$ in $\lambda$
- Useful in many combinatorial arguments

### Hook Lengths

The **hook length** $h(i,j)$ of a box at position $(i,j)$ is:

$$h(i,j) = \lambda_i - j + \lambda'_j - i + 1$$

It equals the number of boxes in the "hook": the box itself, boxes to the right, and boxes below.

**Example:** For $\lambda = (5, 4, 3, 1)$:

```
8 6 5 3 1
6 4 3 1
4 2 1
1
```

**Properties:**
- Hook length formula for number of standard Young tableaux:
  $$f^\lambda = \frac{n!}{\prod_{(i,j)} h(i,j)}$$
- Important in representation theory

## The Partition-Semigroup Bijection

There is a fundamental bijection between:
- **Numerical sets** (sets with finitely many gaps)
- **Integer partitions**

### The Correspondence

Given a numerical set $N$ with gaps $G = \{g_1, \ldots, g_k\}$ (with $g_k = F$ the Frobenius number):

1. Start at $(0, F)$ on a grid
2. Walk to $(F, 0)$ following these rules:
   - If current position is $(x, y)$ with $x + 1 \in G$, move down: $(x, y) \to (x, y-1)$
   - Otherwise, move right: $(x, y) \to (x+1, y)$
3. The resulting path's boundary forms a partition

**Example:** For gaps $\{1, 2, 4, 5, 7\}$:

```
Starting at (0,7), walking to (7,0):
Position 0: gap → down    (0,7) → (0,6)
Position 1: gap → down    (1,6) → (1,5)  
Position 2: gap → down    (2,5) → (2,4)
Position 3: NOT gap → right (3,4) → (4,4)
...
```

This gives partition $(3, 3, 2, 2, 1, 1)$.

### Properties of the Bijection

- **Bijective**: One-to-one correspondence
- **Frobenius preservation**: $F(N) = |\lambda|$ (sum of parts)
- **Genus preservation**: genus$(N)$ = number of boxes in $\lambda$
- **Algorithmic**: Can be computed efficiently

### Why This Matters

The bijection allows us to:
1. **Translate problems**: Convert between domains
2. **Use best algorithms**: Choose representation based on what's efficient
3. **Unify theories**: Connect semigroup theory with partition combinatorics
4. **Visualize**: Use Ferrers diagrams to understand numerical properties

## The Atom Monoid

For a numerical set $N$, the **atom monoid** $\mathcal{A}(N)$ consists of elements that cannot be written as $a + b$ with $a, b > 0$ both in $N$.

### Computing Atoms

From a partition $\lambda$:
- The atoms are related to the hook lengths
- Specifically, $\mathcal{A}(N)$ has gaps corresponding to the flattened hook lengths

**Example:** For $\lambda = (5, 4, 3, 1)$ with hooks:
```
8 6 5 3 1
6 4 3 1
4 2 1
1
```
Flattened: $\{1, 1, 1, 1, 2, 3, 3, 4, 4, 5, 6, 6, 8\}$

### Atom Partition

The **atom partition** $\alpha(\lambda)$ is the partition whose gaps (via the bijection) correspond to the atom monoid of $\lambda$'s gaps.

**Property:** A partition $\lambda$ is a **semigroup partition** if $\lambda = \alpha(\lambda)$.
